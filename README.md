##### **_Achilles CLI Tools for lazy gopher_**

- init go project layout

```bash
achi init
```

- create migration sql file

```bash
achi migration-create
```

- create seeder sql file

```bash
achi seeder-create
```

##### Support OS

- linux (amd64, arm64)
- apple (intel, silicon)
